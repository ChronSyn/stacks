- use `Inline` to render a set of components in a row with equal spacing around them, wrapping onto multiple lines when necessary
- `Inline` supports horizontal alignment of its children
- multiple `Inline` components can be nested to create more complex white space rules

### `space`

- type: `number`
- required: `no`
- default: `0`

::: div doc-example
```tsx
<Inline space={2}>
  <Placeholder height={60} width={90} />
  <Placeholder height={60} width={60} />
  <Placeholder height={60} width={120} />
  <Placeholder height={60} width={200} />
  <Placeholder height={60} width={100} />
  <Placeholder height={60} width={140} />
</Inline>
```

  <img src={useBaseUrl('img/inline-space-1.png')} />
  :::
:::

### `align`

- type: `'left' | 'center' | 'right'`
- required: `no`
- default: `left`
