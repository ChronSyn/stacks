Use the `Column` component along with `Columns`.

:::note

`Column` component accepts [`View`](https://reactnative.dev/docs/view) props.

:::

## Examples

```jsx live
function() {
  return (
    <Columns alignX="between">
      <Column width="content">
        <Placeholder width={100} />
      </Column>
      <Column width="1/2">
        <Placeholder />
      </Column>
      <Column width="content">
        <Placeholder width={100} />
      </Column>
    </Columns>
  )
}
```
