It returns the current window/screen size.

<Tabs {...defaultTabsProps}>
<TabItem value="typescript">

```tsx
declare const useWindowDimensions: () => ReactNative.ScaledSize
```

</TabItem>
<TabItem value="rescript">

```res
let useWindowDimensions: unit => ReactNative.Dimensions.displayMetrics
```

</TabItem>
</Tabs>

## Example

```jsx live
function() {
  const dimensions = useWindowDimensions()

  return (
    <Text>{JSON.stringify(dimensions)}</Text>
  )
}
```
