`Stack` distributes white space evenly between components (vertically).

### `space`

- type: `ResponsiveProp<number>`
- required: `no`
- default: `0`

### `align`

- type: `ResponsiveProp<'left' | 'center' | 'right' | 'stretch'>`
- required: `no`
- default: `stretch`
