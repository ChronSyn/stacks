- `FillObject` covers a very common pattern for creating overlays with position absolute and zero positioning.

```tsx
<FillObject alignX="center" alignY="center" direction="row">
  <Box flex="3/4">
    <Stack space={1}>
      <Placeholder height={60} />
      <Placeholder height={60} />
      <Placeholder height={60} />
      <Placeholder height={60} />
    </Stack>
  </Box>
</FillObject>
```
