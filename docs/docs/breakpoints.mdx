`Stacks` supports the `responsive props` format which allows you to specify an array of values for different screen sizes. Use `responsive props` to customize the spacing, number of columns, or alignments per screen size.

<Tabs {...defaultTabsProps}>
<TabItem value="typescript">

```tsx
type ResponsiveProp<T> = T | Readonly<[T, T]> | Readonly<[T, T, T]>
```

</TabItem>
<TabItem value="rescript">

```res
type responsiveProp<'a> = array<'a>
```

</TabItem>
</Tabs>

There are three available breakpoints: `mobile` (default value, since `Stacks` components are mobile-first), `tablet` and `desktop`.

<Tabs {...defaultTabsProps}>
<TabItem value="typescript">

```tsx
type Breakpoint = 'mobile' | 'tablet' | 'desktop'
```

</TabItem>
<TabItem value="rescript">

```res
type breakpoint = [#mobile | #tablet | #desktop]
```

</TabItem>
</Tabs>

## Provider

You can define custom breakpoints in the provider.

<Tabs {...defaultTabsProps}>
<TabItem value="typescript">

```tsx
import { StacksProvider } from '@mobily/stacks'

const App = () => {
  return (
    <StacksProvider spacing={4} breakpoints={{ tablet: 762, desktop: 992 }}>
      …
    </StacksProvider>
  )
}
```

</TabItem>
<TabItem value="rescript">

```res
open Stacks

@react.component
let make = () =>
  <StacksProvider spacing=4. breakpoints={tablet: 792., desktop: 996.}>
    …
  </StacksProvider>
```

</TabItem>
</Tabs>

## Examples

If you want small spacing on mobile but medium spacing from tablet upwards.

<Tabs {...defaultTabsProps}>
<TabItem value="typescript">

```tsx
<Stack space={[1, 4]}>
  …
</Stack>
```

</TabItem>
<TabItem value="rescript">

```res
<Stack space={[1., 4.]}>
  …
</Stack>
```

</TabItem>
</Tabs>

If you want top alignment on mobile and center on tablet upwards.

<Tabs {...defaultTabsProps}>
<TabItem value="typescript">

```tsx
<Columns space={1} alignY={['top', 'center']}>
  <Column>
    …
  </Column>
  <Column>
    …
  </Column>
</Columns>
```

</TabItem>
<TabItem value="rescript">

```res
<Columns space=[1.] alignY=[#top, #center]>
  <Column>
    …
  </Column>
  <Column>
    …
  </Column>
</Columns>
```

</TabItem>
</Tabs>

If you want to stack columns vertically on smaller screens, you can provide the `collapseBelow` prop.

<Tabs {...defaultTabsProps}>
<TabItem value="typescript">

```tsx
<Columns space={1} collapseBelow="tablet">
  <Column>
    …
  </Column>
  <Column>
    …
  </Column>
</Columns>
```

</TabItem>
<TabItem value="rescript">

```res
<Columns space=[1.] collapseBelow=#tablet>
  <Column>
    …
  </Column>
  <Column>
    …
  </Column>
</Columns>
```

</TabItem>
</Tabs>
