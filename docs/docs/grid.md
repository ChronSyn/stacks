`Grid` renders a design system grid. Place it only once, at the top of your react tree.

[[ codesandbox id="grid-0hc8m" ]]

### `columns`

- type: `number`
- required: `no`
- default: `8`

### `margin`

- type: `number`
- required: `no`
- default: `1`

### `gutter`

- type: `number`
- required: `no`
- default: `1`

### `opacity`

- type: `number`
- required: `no`
- default: `0.1`
